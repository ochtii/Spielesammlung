Über das Projekt

Projektname: Österreich Spielesammlung

Version: Wird geladen...

Letztes Update: Wird geladen...

Repository: GitHub

Eine interaktive Web-Anwendung zum Lernen österreichischer Geographie, Hauptstädte, Kennzeichen und mehr.

Verwendete Technologien

HTML5 CSS3 JavaScript ES6+ LocalStorage API Font Awesome Wikimedia Commons

Hauptfunktionen

  • 5 Spielmodi: Hauptstädte, Kennzeichen, Einwohner-Vergleich, Wappen und Welthauptstädte
  • Punktesystem: Detaillierte Statistiken mit Rang-System und Erfolgen
  • Responsive Design: Optimiert für Desktop, Tablet und Mobile
  • Dark Mode: Automatische Anpassung an System-Einstellungen
  • Anpassbar: Umfangreiche Einstellungsmöglichkeiten
  • Offline-fähig: Funktioniert ohne Internetverbindung

Datenschutz & Speicherung

  • Lokale Speicherung: Alle Daten werden ausschließlich lokal im Browser gespeichert
  • Keine Server-Kommunikation: Es werden keine Daten an externe Server übertragen
  • Kein Tracking: Keine Analyse-Tools oder Tracking-Scripte
  • Volle Kontrolle: Daten können jederzeit über die Entwickler-Tools gelöscht werden

Browser-Kompatibilität

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

Danksagungen

  • Wappen-Bilder: Wikimedia Commons
  • Icons: Font Awesome
  • Schriftarten: Google Fonts
